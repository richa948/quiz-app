# Quiz App

A simple and interactive quiz application built using React and JavaScript. Users can answer multiple-choice questions, navigate through the quiz, and view their final score at the end.

## Features

* Multiple-choice questions
* Instant answer selection
* Automatic navigation to the next question
* Score calculation
* Responsive and clean UI
* Deployed on Vercel

## Tech Stack

* React.js
* JavaScript
* CSS
* Vercel

## Installation

1. Clone the repository:

```bash
git clone <your-repository-url>
```

2. Navigate to the project folder:

```bash
cd quiz-app
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```
